Manual de Implantação

Este guia orienta o implantador na configuração do ambiente completo para o FILA-APP, desde a instalação das dependências até a inicialização do sistema.

O sistema foi projetado para ser "Auto-Instalável", exigindo apenas que os pré-requisitos básicos estejam presentes no servidor.

Preparação

Localize a pasta INSTALADORES no diretório raiz do projeto. Nela você encontrará os arquivos necessários:

  • Node.js: Necessário para rodar o servidor de aplicação.
  • MySQL: Necessário para o armazenamento persistente de dados.

1. Instalando Node.js

O Node.js é o motor do sistema. Sem ele, a aplicação não inicia.

1 Execute o instalador do Node.js (v16 ou superior).

2 Avance todas as etapas mantendo as configurações padrão.

3 Ao finalizar, abra um terminal (CMD) e digite node -v para confirmar.

2. Instalando MySQL

O banco de dados armazena as senhas, usuários e métricas.

1 Execute o instalador do MySQL (Recomendado: MySQL Community Server 8.0).

2 Configure a senha do usuário root (você precisará dela no Wizard).

3 Certifique-se de que o serviço está em execução.

3. Wizard de Instalação

Com os componentes instalados, o sistema está pronto para ser configurado.

1 Execute o arquivo 3-START.bat.

2 Acesse http://localhost:3000 no seu navegador.

3 O sistema detectará que não está configurado e abrirá o Wizard de Instalação.

Destaques do Wizard:
  • Verificação automática de componentes.
  • Criação automática do banco de dados e tabelas.
  • Geração de perfis e dados de teste (opcional).

4. Pós-Instalação

Após concluir o Wizard, o sistema estará pronto para uso imediato.
