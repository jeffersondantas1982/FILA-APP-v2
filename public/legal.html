2. Resumo Funcional (INPI Area)

"O FILA-APP é um software de gerenciamento de filas de atendimento em tempo real, desenvolvido em arquitetura cliente-servidor (Node.js). O sistema utiliza comunicação via WebSockets (Socket.io) para sincronização instantânea entre a emissão de senhas (Totem), a gestão de chamadas (Painel do Operador) e a visualização pública (Painel de TV). Inclui módulos de administração para controle de setores, permissões de usuários via RBAC e geração de relatórios estatísticos de produtividade. O armazenamento de dados é estruturado em MySQL com persistência de auditoria."
