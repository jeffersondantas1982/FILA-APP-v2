# Guia de Contribuição 🤝

Obrigado por se interessar em contribuir para o **FILA-APP**! Como um projeto mantido pela equipe de TI do Hospital Presidente Vargas e licenciado sob a GPL v3, valorizamos muito o apoio da comunidade para tornar o sistema mais eficiente e acessível.

## 🚀 Como posso contribuir?

### 1. Relatando Bugs
- Verifique se o bug já não foi relatado nas *Issues*.
- Se não, abra uma nova issue descrevendo detalhadamente o problema, passos para reproduzir e comportamento esperado.

### 2. Sugerindo Melhorias
- Tem uma ideia para uma nova funcionalidade ou melhoria na UI? Abra uma issue de sugestão!

### 3. Enviando Pull Requests
1. Faça um **Fork** do projeto.
2. Crie uma **Branch** para sua modificação (ex: `git checkout -b feature/minha-melhoria`).
3. Commit suas mudanças com mensagens claras.
4. Faça o **Push** para sua branch.
5. Abra um **Pull Request** para a branch principal (`main`).

## 📜 Código de Conduta

Ao participar deste projeto, você concorda em manter um ambiente respeitoso, profissional e colaborativo. Assédio, discriminação ou comportamento ofensivo de qualquer tipo não serão tolerados.
